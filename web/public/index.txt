Easy and fast file sharing from the command line.

Upload a file:

  $ curl --upload-file ./hello.txt {{.WebAddress}}hello.txt
  {{.WebAddress}}{{.SampleToken}}/hello.txt

Authenticated upload (bigger files, longer retention):

  $ curl --user <user>:<password-or-api-token> --upload-file ./big.iso {{.WebAddress}}big.iso

Per-upload limits via headers:

  curl -H "Max-Days: 7"      ...      # auto-delete after N days (capped per tier)
  curl -H "Max-Downloads: 1" ...      # cap the download count

Download:

  $ curl {{.WebAddress}}<token>/<filename> -o <filename>

Delete (token returned in X-Url-Delete response header):

  $ curl -X DELETE {{.WebAddress}}<token>/<filename>/<delete-token>
